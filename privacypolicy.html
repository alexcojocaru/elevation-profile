TL;DR: there is nothing to worry about.

The data you upload on the page DOES NOT leave your browser.

All the data processing and rendering is done client side.

The page uses live OpenStreetMap data to update the map as you move or zoom.

There are four Javascript libraries used on the page:

  • Leaflet - to render the map and the track(s)
  • Leaflet.Heightgraph - to draw the elevation chart
  • togeojson - to convert between various geo data formats
  • geo-data-exchange - more geo data format conversion

There is no tracking, telemetry, analytics, unless done by the OpenStreetMap provider without my knowledge.
